# CSS 样式

## css 背景

```js
// 背景设置为灰色
p {background-color: gray;}
// 设置背景图片
body {
  background-image: url(/i/eg_bg_03.gif);
  background-repeat: repeat-y;
  }
```

## CSS 文本

```js
// 所有段落的首行缩进 5 em
p {text-indent: 5em;}
// 段落缩进可以使用负值
p {text-indent: -5em; padding-left: 5em;}
// 段落缩进可以使用百分比值
p {text-indent: 20%;}
```
